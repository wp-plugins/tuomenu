== Description ==

Inserisci il menu della tua scheda direttamente sul tuo blog. Includi il tag [tuomenu url="url-tuo-ristorante"] nella pagina desiderata, tutto qui.
Tuomenu è un sito gratuito che ti permette di creare una scheda del tuo ristorante e registrare il tuo menu online. Il menu diventa poi ricercabile e condivisibile con i socialnetwork.
Registrati ora su www.tuomenu.com, poi installa il plugin sul tuo sito/blog ed il menu del tuo ristorante apparirà senza fatica sul sito del tuo ristorante.

== Installation ==

L'installazione del plugin è semplicissima


1. carica la cartella `tuomenu` dentro la directory `/wp-content/plugins/` del tuo sito/blog
1. Attiva il plugin dal pannello di controllo di Wordpress
1. Inserisci [tuomenu url="url-tuo-ristorante"] nella pagina dove vuoi mostrare il tuo menu.
